How to use it?

This is an inline calculator, each calculation will be calculated by line.

1 + 1 will return 2

Operator Function
+ Add
- Subtract
* Multiply
/ Divide
^ Power

To see more possible operations, to go mathjs operators documentation.

Comments

If you want to create a comment, just use the # sign, everything after the # will be ignored.

Variables

You can assign variables to use later.

pizzas = 2

pizzaPrice = 30

people = 4

(pizzas * pizzaPrice) / people will return 15
